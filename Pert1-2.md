## 1. Panduan Persiapan (Getting Started)

Sebelum memulai praktikum, pastikan mahasiswa telah menyiapkan lingkungan kerja Python.

### Langkah-langkah Memulai:
1. **Instalasi:** Unduh Python versi terbaru di [python.org](https://www.python.org/). Pastikan mencentang **"Add Python to PATH"** saat instalasi.
2. **Verifikasi:** Buka Terminal/CMD, ketik: `python --version`.
3. **Editor:** Gunakan VS Code atau IDE lainnya. Simpan file dengan ekstensi `.py`.
4. **Eksekusi:** Jalankan program dengan perintah: `python nama_file.py`.
